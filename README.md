# Samuel Jarai's — Portfolio

Personal portfolio site built with Vue 3 + TypeScript and TailwindCSS, deployed to GitHub Pages via CI/CD.

![Portfolio Preview](public/images/profile_pic.png)

## Run Locally

```bash
git clone https://github.com/jaggerjack61/portfolio.git
cd portfolio
npm install
npm run dev
```
