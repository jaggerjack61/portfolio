# Samuel Jarai - Developer Portfolio

A professional, minimalistic single-page developer portfolio website built with **Vue.js 3** and **TailwindCSS**. This site showcases my full-stack engineering skills, experience, and projects through a clean, modern UI design.

### Prerequisites

*   Node.js (v16.0.0 or higher)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/jaggerjack61/portfolio.git
    cd portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run build
    ```
